# VimStudyMemo
A Study Memo of Vim

## 1. [>], [f], [\*], [\\],[A]
| Command | Effect | Repeat |
|:-------:|:------:|:------:|
|>        | Add a progression  |    .   |
|f{char}  | Search a character |    ;   |
|\*       | Search a word      |    n   |
|\\partern| Search a partern   |    n   |
|A        | Move the the end of line and Change to Insert Mode| . |
